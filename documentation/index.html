AngularJS in action documentation

ng-click


gets triggered when the mouse clicks the object.
 

ng-change


gets triggered when object is changed (dropdown, textbox etc).
 

ng-focus / ng-blur


{{focusObj.dropdown}} gets triggered when object gets focus (with tab key or mouse click).
{{focusObj.input}}

ng-copy


gets triggered when the user tries to copy (CTRL-C).

ng-cut


gets triggered when the user tries to copy (CTRL-X).

ng-paste


gets triggered when the user tries to paste (CTRL-V).

ng-dblclick


gets triggered when the mouse clicks the object.
